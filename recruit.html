암호최적화 및 응용 연구실(Crypto Optimization & Application Lab)에서 열정적인 석사 과정 및 학부 연구원생을 모집합니다.

관심있는 학생은 scseo@kookmin.ac.kr로 관심연구분야와 자기소개 적어 이메일로 지원하시기 바랍니다 (이후, 면담을 통해 선발).

연구분야

  • Optimization of cryptographic algorithms

  • Applied cryptography

  • IoT Security

  • Embedded Security

  • Network Security

  • CMVP (Cryptographic Module Validation Program)

연구내용

  • 임베디드 환경에서 암호 알고리즘 최적화 연구 (8-bit, 16-bit, 32-bit 임베디드 장치에서 암호 최적화)

  • PQC (Post Quantum Cryptography) 최적 구현 연구

  • 병렬 컴퓨팅 기술(GPU, CPU SIMD, OpenMP 등)을 이용한 암호알고리즘 고속 구현 기술 연구

  • IoT 보안 기술 연구

  • 암호 응용 기술 연구 (암호 기반 네트워크 보안 기술)

  • CMVP (Cryptographic Module Validation Program) 시험기술 개발 연구

자격요건

정보보호분야에 대한 관심이 있어야 하며, 아래 사항에 대해 최소 하나의 지식을 보유해야 함

  • 암호학

  • 프로그래밍 능력 (C 또는 C++ 또는 Java 또는 어셈블리언어)

  • 운영체제, 컴퓨터구조 이해

  • 인터넷 프로토콜 이해

대상

  • 석사과정 (학사 졸업 예정 자)

  • 학부 연구원생 (3~4학년 재학생)
